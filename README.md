# Task Management Frontend (Next.js + TypeScript)

A frontend built using Next.js (App Router) with TypeScript for the Task Management system.

## Tech Stack
- Next.js 14+
- TypeScript
- Axios
- Tailwind CSS
- React Hooks


## Environment Variables (.env.local)

NEXT_PUBLIC_API_URL="http://localhost:5000


## Installation
```bash
git clone git@github.com:786Ayush/task-management-frontend.git
cd task-management-frontend
npm install
```
Run Dev Server
```bash
npm run dev
```

Build for Production
```bash
npm run build
npm start
```
